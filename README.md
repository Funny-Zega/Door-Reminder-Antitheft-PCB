# Door Reminder and Anti-Theft Circuit

A logic-based door reminder and anti-theft alarm circuit designed in Altium. This project was developed as Lab 1 for the Electrical Electronic Circuits course at Ho Chi Minh City University of Technology (HCMUT).

## Project Overview
The idea for this project stems from the common problem of doors being left open accidentally, leading to wasted energy and unwanted pests. If a door remains open for too long, it also becomes a security concern. 

This circuit offers a simple, cost-effective solution using basic components like sensors, timers, and logic ICs to alert users when the door is left open, while also functioning as an anti-theft system in case of unauthorized access.

## Features
* **Dual Modes**: Functions as both a door reminder and an anti-theft alarm.
* **Adjustable Delay**: Uses an LM555 timer in monostable mode to generate a ~5.17s delay before triggering the reminder alarm.
* **Anti-Theft Mode**: Instantly activates the buzzer continuously when a break-in is detected.
* **Hardware Debouncing**: Implements an SN74LS14N Schmitt Trigger to eliminate noise and stabilize the door sensor signal.
* **Custom IR Sensor**: Includes an independent, custom-built Infrared (IR) sensor module using an IR LED, Photodiode, and LM358 Op-Amp.

## Hardware Components
The system is divided into two independent modules:

**1. Main Logic Circuit**
* `SN74LS74AN`: Flip-Flop to store the sensor state and control the buzzer.
* `LM555 Timer`: Generates the delay signal.
* `SN74LS14N`: Schmitt Trigger for noise filtering.
* `SN74LS08N`: AND gate to handle the anti-theft mode logic.
* `Buzzer`: 6V alert mechanism.

**2. Infrared Sensor Module**
* `IR LED & Photodiode`: Emitter and receiver pair.
* `Transistor 2N2222`: Amplifies the small current generated by the photodiode.
* `LM358`: Operational amplifier to output the final digital signal based on light intensity.
* `10kΩ Potentiometer`: Adjusts the sensitivity of the LM358.

## PCB Design Specifications
Designed in Altium with the following Design Rules:
* **Routing Layer**: Bottom layer used for routing to optimize space and separate power from signal lines.
* **Clearance**: Minimum 10 mil (IR Sensor) and 20 mil (Main PCB) to prevent electrical shorts.
* **Trace Width**: Minimum 20 mil, Preferred 25 mil, Maximum 30 mil (for signal integrity and current handling).
* **Ground Pour**: Applied on the bottom layer to reduce EMI.
